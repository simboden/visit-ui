# visit-ui

visit-ui is an opinionated UI library for building Svelte 5 webapp frontends.

Live demo: https://simboden.github.io/visit-ui/

## Installation

```bash
pnpm add visit-ui
```

## Styles

Import the shared theme styles once in your app's global CSS:

```css
@import 'tailwindcss';
@import 'visit-ui/theme.css';
```

## What it provides

- Theming driven by CSS custom properties and a runtime theme store
- Application layout primitives such as AppShell, AppNavBar, and sidebars
- Responsive UI components built for Svelte 5

## Peer dependencies

- svelte
- @sveltejs/kit
